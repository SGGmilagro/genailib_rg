OpenAI Chatbot Library Source Code Using OpenAI API

Welcome to the documentation for genailib_rg! We are excited to have you here.

This library allows you to interact with OpenAI’s chat API with ease.

Make sure to set your .env file up with an appropriate openAI API code and make sure you know that this is a strictly private key! You can not have it leaked to github or dokku!

Visit different pages to expereince what our chatbot can do! Yes, we know it is not as advanced but still, enjoy!
